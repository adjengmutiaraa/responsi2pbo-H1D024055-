Pertanyaan Refleksi
Setelah menyelesaikan kode, jawab pertanyaan berikut:

1. Apa fungsi keyword 'extends' dalam deklarasi class Java?
2. Jelaskan perbedaan antara Method Overloading (yang ada di latihan sebelumnya) dengan Method Overriding (yang ada di latihan ini)!
3. Apa kegunaan keyword 'super' di dalam constructor subclass (Manajer)?
4. Jika variabel di class Karyawan diubah access modifier-nya menjadi 'private', apa yang akan terjadi pada class Manajer? Bagaimana solusinya?
5. Mengapa kita perlu menggunakan Inheritance? Apa keuntungan utamanya dibandingkan membuat dua class yang terpisah sama sekali (misal class Karyawan dan class Manajer dibuat terpisah tanpa hubungan)?

Jawaban:
1. Untuk menunjukkan bahwa class tersebut merupakan subclass dari class lain. Jadi class tersebut dapat mengimplementasikan
   atribut dan method yang ada di class lain (superclass)
2. - Overloading: terjadi dalam class yang sama, nama method sama tetapi parameter berbeda (jumlah atau tipe berbeda)
   - Overriding: terjadi antara parentâ€“child, nama method dan parameternya sama, tetapi subclass membuat implementasi baru untuk menggantikan perilaku parent
3. super(...) digunakan untuk memanggil constructor parent (Karyawan)
4. Jika variabel pada class Karyawan diubah menjadi private, maka class Manajer sebagai subclass tidak dapat lagi
   mengakses atribut tersebut secara langsung dan akan menyebabkan error pada baris kode di Manajer yang mencoba menggunakan atribut parent.
   Solusinya, atribut tetap dibuat protected atau dengan membuat method getter/setter pada class Karyawan agar subclass
   tetap bisa mengunakan atribut tersebut.
5. Inheritance memungkinkan kode untuk bisa digunakan lagi, mengurangi duplikasi, dan membuat struktur program lebih ringkas.
   Subclass dapat memperluas fungsi parent tanpa menulis ulang atribut atau method dasar. Jika class dibuat terpisah tanpa inheritance,
   akan ada banyak kode yang harus disalin ulang, sulit dipelihara, dan perubahan di satu class tidak otomatis diterapkan ke class lain.
