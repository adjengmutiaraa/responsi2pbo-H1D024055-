Pertanyaan Refleksi
Setelah menyelesaikan kode, jawab pertanyaan berikut:

Jelaskan perbedaan antara fungsi (function) dan prosedur (procedure) dalam pemrograman berorientasi objek.
Mengapa penting menggunakan konstanta (final variables) dalam suatu kelas? Apa keuntungannya?
Apa yang terjadi jika kita mencoba mengubah nilai suatu konstanta setelah objek dibuat? Berikan contoh.
Bagaimana cara mendefinisikan konstanta dalam bahasa Java?

Jawaban
1. Dalam PBO, fungsi (function) akan mengembalikan nilai (return value) dengan tipe data tertentu seperti int,
   double, dll dan biasanya digunakan untuk melakukan perhitungan dan menghasilkan output.
   Sedangkan prosedur (procedure) tidak mengembalikan nilai (void) dan biasanya digunakan untuk
   menampilkan output dan menjalankan rangkaian perintah
2. Penggunaan konstanta (final variables) penting dalam suatu kelas karena nilai tidak dapat diubah setelah didefinisikan.
   Keuntungan penggunaan konstanta dalam class:
   a. Dapat menyimpan nilai yang tidak bisa diubah setelah didefinisikan (nilai tetap), jadi dapat
      menghindari perubahan yang tidak disengaja dan dapat menyebabkan bug
   b. Membuat kode lebih mudah dibaca dan dipahami
   c. Lebih mudah untuk memelihara kode dan dapat digunakan kembali di berbagai metode
3. Program tidak dapat berjalan dengan benar (error). Contoh:
   class Pengguna {
       public static final double faktorAktivitas = 1.2;
       public static void main(String[] args) {
           p1.faktorAktivitas = 1.5;
       }
   }
   Kode tersebut akan error ketika dirun dan akan menampilkan output
   "cannot assign a value to static final variable faktorAktivitas"
4. Cara mendefinisikan konstanta dalam java:
   access_modifier static final tipe_data nama_konstanta = nilai;
